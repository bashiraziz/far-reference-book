# 53.102 Current editions

The form prescriptions in subpart  [53.2](/part-53/53.2)  and the FAR forms located at https://www.gsa.gov/forms

contain current edition dates. Contracting officers  shall  use the current editions unless otherwise

authorized under this regulation.

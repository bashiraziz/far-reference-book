# 53.235 Research and development contracting (SF 298)

SF 298 (2/89), Report Documentation Page . SF 298  is prescribed for use in submitting scientific and

technical reports to contracting officers  and to technical information libraries, as specified in [35.010](/part-35/35.010)

and ANSI Standard Z39.18.

# 15.407-3 Forward pricing rate agreements

(a) When certified cost or pricing data  are required, offerors  are required to describe any forward

pricing rate agreements  (FPRAs) in each specific pricing  proposal to which the rates apply and to

identify the latest cost or pricing data  already submitted in accordance with the FPRA. All data

submitted in connection with the FPRA, updated as necessary, form a part of the total data that the
